ОФЛАЙН-ПРИЛОЖЕНИЕ УЧЕТА СКЛАДСКИХ СБОРЩИКОВ

1) Запуск приложения
- Установите Python 3 (если не установлен).
- В папке проекта выполните:
  py -3 -m pip install -r requirements.txt
  py -3 main.py

WEB-ВЕРСИЯ ДЛЯ iPHONE
- Запуск локально:
  py -3 -m uvicorn app_web:app --host 0.0.0.0 --port 8000
- На iPhone откройте в Safari:
  http://IP_ВАШЕГО_ПК:8000
- Добавление на экран iPhone:
  Safari -> Поделиться -> На экран Домой
- Для размещения на бесплатном хостинге используйте:
  start command: uvicorn app_web:app --host 0.0.0.0 --port $PORT

2) Что внутри
- Ввод данных: добавление записи по сборщику.
- Журнал: просмотр записей, фильтры, экспорт Excel/PDF, печать.
- Статистика: за день, за период, общий объем и по типам работ.

3) Где хранятся данные
- Локальная база: data\warehouse.db
- Экспорт Excel: exports\
- Файлы печати/PDF: prints\

4) Сборка в EXE
- Запустите build_exe.bat
- Готовый файл: dist\warehouse_app.exe

5) Формат даты
- Используйте YYYY-MM-DD (пример: 2026-04-07)
